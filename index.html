Display

Existem 'n' displays

Display grid: é perfeito para criar layouts mais complexos, como a página de um jornal, onde precisa-se de várias linhas e colunas

Display Flex: ele organiza elementos, quando aplicado esses elementos se tornam uma flex container, a partir dai é poss

Display FlexBox:é um módulo completo e não uma única propriedade; algumas delas devem ser declaradas no container (o elemento-pai, que chamamos de flex container), enquanto outras devem ser declaradas nos elementos-filhos (os flex itens).
